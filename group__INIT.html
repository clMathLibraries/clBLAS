clblasStatus clblasSetup ( void  )

Initialize the clblas library.

Must be called before any other clblas API function is invoked.

Note
This function is not thread-safe.
Returns
  • clblasSucces on success;
  • clblasOutOfHostMemory if there is not enough of memory to allocate library's internal structures;
  • clblasOutOfResources in case of requested resources scarcity.

void clblasTeardown ( void  )

Finalize the usage of the clblas library.

Frees all memory allocated for different computational kernel and other internal data.

Note
